IPv4 Subnetting Program

What this app is about?

This app can be used to check your work whenever your're learning or practicing IPv4 subnetting. This app is free for anyone to use.

How to use this app?

  1. Enter an IPv4 Address.
  2. Enter a Subnet Mask.
  3. Enter the number of bits (or borrowed bits) for subnet portion.
  4. Click the submit button.

About the output?

  • By default the app will display the first subnet (Subnet zero).
  • You can find other subnets if available by clicking on the subnet label or the subnet number input next to it.
  • The Hosts label represents the number of host per subnet.
